# Proyecto Final JavaScript
### Nombre: Carlos Ferrer
### Cedula: 28.326.446
### Sección: N1113

#### Descripcion

Sistema Administrativo para la recoleccion de información sobre los clientes y mantenimientos de los vehiculos de un taller mecánico. El sistema fue realizado utilizando NodeJs con su Framework express y MongoDB para el backend a través de su servicio en la nube MongoDB Atlas. Para el diseño se utilizo Bootstrap como framework principal.
